# Learning Java

Учебный репозиторий с решениями алгоритмических задач.

## Структура:
- arrays — задачи на массивы
- backtracking — задачи с перебором
- sliding-window — задачи со скользящим окном
- bit-manipulation — побитовые операции
- hash-table — хеш-таблицы

## О себе
Изучаю Java и алгоритмы, регулярно решаю задачи и прокачиваю навыки решения задач и написания чистого кода.
